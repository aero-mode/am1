Android
=
Настройка через v2rayNG
-

1. Скачайте apk файл с диска 
| Файл                                | Для какой архитектуры              | Где используется                                                                |
| ----------------------------------- | ---------------------------------- | ------------------------------------------------------------------------------- |
| **v2rayNG_1.10.26_arm64-v8a.apk**   | ARM **64-бит**                     | большинство современных Android-телефонов (2018+), Snapdragon, MediaTek, Exynos |
| **v2rayNG_1.10.26_armeabi-v7a.apk** | ARM **32-бит**                     | старые телефоны, Android 5–8, дешёвые модели                                    |
| **v2rayNG_1.10.26_universal.apk**   | Универсальная сборка (всё в одном) | подойдёт для любого устройства, но весит больше                                 |
| **v2rayNG_1.10.26_x86.apk**         | x86 (32-бит)                       | редкие планшеты, эмуляторы, Intel-устройства                                    |
| **v2rayNG_1.10.26_x86_64.apk**      | x86_64 (64-бит Intel)              | эмуляторы Android (BlueStacks, LDPlayer и др.)                                  |
